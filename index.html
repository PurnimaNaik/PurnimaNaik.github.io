Projects
Freshers’ Guide Website
Created a website that provided information about mobile plans and banks to the new students of Northeastern University. Features include real time profile simulation, bank recommending calculator, pagination, graphs, animations and email functionality.

E-Commerce Cars Website
Developed an e-commerce application to compare, buy and sell cars. Implemented the project using annotation based controllers, annotation based mapping, inheritance using table per subclass mapping, spring security and web flow. Integrated cookies and hidden form field to track user clicks and show customized content.

High School Database Management and BI
Cleaned and normalized a high school database. Created entity-relationship and dimensional models with respect to various factors such as attendance, discipline, academic performance etc. Developed data cubes and data visualizations for student, parent, teacher and administrator roles. Formulated SQL queries for the dashboard with respect to the aforementioned roles.

Enterprise Management Application
Designed & developed a java application for companies that manufacture chemically synthesized products. The project featured functionalities for 7 different use cases. Using singleton pattern, implemented the concept of logins, work requests and work queues to manage the demand and supply of raw materials and finished products. Displayed simulation of factory workspace and sensor dependent manufacturing processes using timers.
